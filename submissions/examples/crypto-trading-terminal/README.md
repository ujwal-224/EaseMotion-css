# Animation: Crypto Trading Terminal

A professional trading dashboard inspired by Binance, Coinbase, and TradingView. It combines a market watchlist, trading chart, portfolio overview, and trading controls into a modern interface.

## Features

- Pure HTML & CSS
- Market watchlist
- Trading chart visualization
- Buy/Sell panel
- Portfolio summary
- Responsive layout
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="asset">

    <strong>BTC</strong>

    <span class="up">+2.4%</span>

</div>
```

## Why is it useful?

Trading terminals are among the most information-dense interfaces in modern applications. This example demonstrates how to organize financial data, controls, and visualizations into a clean and accessible layout while maintaining a professional appearance.
