# Collapsible Sidebar Navigation Example

1. What does this do? This example demonstrates a reusable collapsible sidebar navigation pattern for dashboard and admin interfaces that smoothly transitions between full and icon-only states.
2. How is it used? Apply `.ease-sidebar-layout` to the main container and toggle the `.ease-sidebar-collapsed` class on it to expand or collapse the sidebar.
3. Why is it useful? It helps developers quickly implement responsive, space-saving navigation for dashboards using EaseMotion's built-in utilities while maintaining smooth animations and mobile compatibility.
