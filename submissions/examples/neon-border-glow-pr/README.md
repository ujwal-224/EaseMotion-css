# Neon Border Glow

## What does this do?
An animated pulsing border glow using CSS `box-shadow` to create a
modern neon lighting effect for cards, buttons, and highlighted
components.

## How is it used?

```html
<div class="neon-glow-card">
  Neon Glow
</div>
```

**Color variants:**
```html
<div class="neon-glow-card neon-glow-card-pink">Pink Neon</div>
<div class="neon-glow-card neon-glow-card-green">Green Neon</div>
```

## Why is it useful?
This effect complements modern dark UI themes and provides developers
with a reusable CSS-only glow animation that fits EaseMotion CSS's
animation-first philosophy. Pure CSS, no JS required, with full
prefers-reduced-motion support.
