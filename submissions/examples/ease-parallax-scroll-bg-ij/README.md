# Parallax Scroll BG

Parallax scrolling background with layers moving at different speeds via --psb-speed.

## Features

- Multiple background layers
- Parallax speed via --psb-speed
- Smooth scroll-based transform
- Depth effect on scroll

## Usage

Set --psb-speed (0-1) on each .psb-layer. JS sets --psb-offset based on scroll position.
