# Ease Progress Circle Dash

A smooth animated circular progress indicator created using HTML, CSS, and SVG. The progress ring uses `stroke-dasharray` and `stroke-dashoffset` to produce a clean easing animation.

## Features

- Smooth CSS animation
- SVG-based circular progress
- Responsive centered layout
- Pure HTML & CSS
- Easy to customize

## Files

```
ease-progress-circle-dash-iij/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in any modern web browser.
2. The progress circle will animate automatically.

## Technologies Used

- HTML5
- CSS3
- SVG
