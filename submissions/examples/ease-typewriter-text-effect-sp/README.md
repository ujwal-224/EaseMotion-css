# EaseMotion Typewriter Text Effect

## What does this do?

This component creates a smooth typewriter animation with a blinking cursor.

## How is it used?

```html
<div class="typewriter">
    <span id="type-text"></span>
    <span class="cursor">|</span>
</div>
```

## Why is it useful?

It helps highlight important text on landing pages, hero sections, portfolios, and other websites with an engaging typing animation.
