# Calendar Date Picker

Interactive calendar date picker with hover scale, today pulse ring, and selected fill animation.

## Features

- Calendar grid with day cells
- Hover: scale(1.1) with spring transition
- Selected date: background fill animation via `@keyframes cdp-fill-select`
- Today's date: pulse ring via `@keyframes cdp-pulse-ring`
- JS click toggles `.cdp-selected` class
- Google Fonts Inter (400, 600, 700)

## Usage

Click any date cell to select it. The selected cell animates a background fill. Hovering scales the cell up. The current date displays a pulsing ring.
