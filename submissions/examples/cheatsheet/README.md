# EaseMotion CSS Class Reference Cheatsheet

> Searchable single-page reference for every ease-* class

## What it does

A fully searchable, filterable cheatsheet listing every ease-* class
with its description and category. Helps beginners find the right
class without digging through source files.

## Features

- Live search filtering by class name or keyword
- Category filters: Animation, Hover, Component, Utility, Layout
- Shows count of visible results
- Sticky search bar while scrolling
- Color-coded category tags
- Zero dependencies, pure HTML + CSS + vanilla JS

## Usage

Open demo.html in any browser. No build step, no server needed.

## Categories covered

| Category | Examples |
|---|---|
| Animation | ease-fade-in, ease-slide-up, ease-typewriter, ease-shake |
| Hover | ease-hover-grow, ease-hover-lift, ease-shake-hover |
| Component | ease-btn, ease-card, ease-progress |
| Utility | ease-glass, ease-truncate, ease-text-gradient |
