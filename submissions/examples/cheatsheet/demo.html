EaseMotion CSS

Class Reference Cheatsheet -- search and filter every ease-* class

Animation
ease-fade-in
Fades element in from transparent to fully visible on page load.
Animation
ease-slide-up
Slides element upward while fading in on load.
Animation
ease-slide-down
Slides element downward while fading in on load.
Animation
ease-slide-left
Slides element in from the right on load.
Animation
ease-slide-right
Slides element in from the left on load.
Animation
ease-zoom-in
Scales element up from 0.8 while fading in.
Animation
ease-bounce
Bouncy entrance animation with overshoot effect.
Animation
ease-flip
3D flip entrance animation on the Y axis.
Animation
ease-typewriter
Pure CSS typewriter effect with blinking cursor. Set --ease-typewriter-steps to char count.
Animation
ease-typewriter-fast
Faster typing speed variant for ease-typewriter.
Animation
ease-typewriter-slow
Slower typing speed variant for ease-typewriter.
Animation
ease-shake
Horizontal shake for error feedback. Add/remove class via JS to retrigger.
Animation
ease-wobble
Rotational wobble for playful attention effects.
Animation
ease-pulse-attention
Scale pulse for heartbeat-style attention animations.
Animation
ease-reveal
Scroll-triggered fade-up. Add is-visible via IntersectionObserver JS.
Animation
ease-reveal-left
Scroll-triggered slide from left.
Animation
ease-reveal-right
Scroll-triggered slide from right.
Animation
ease-reveal-zoom
Scroll-triggered zoom-in entrance.
Utility
ease-delay-100 to ease-delay-500
Animation delay helpers: 100ms, 200ms, 300ms, 400ms, 500ms. Use for staggered entrances.
Hover
ease-hover-grow
Scales element up slightly on hover.
Hover
ease-hover-lift
Lifts element with shadow on hover.
Hover
ease-hover-glow
Adds glowing box-shadow on hover.
Hover
ease-hover-shimmer
Shimmering light sweep across element on hover.
Hover
ease-shake-hover
Shakes element on hover. No JS required.
Hover
ease-wobble-hover
Wobbles element on hover. No JS required.
Component
ease-btn
Base button component with padding, radius, and transition.
Component
ease-btn-primary
Primary filled button using --ease-color-primary.
Component
ease-btn-outline
Outlined button with border, transparent background.
Component
ease-card
Surface card with border, padding, and border-radius.
Component
ease-progress
Progress bar track. Use with ease-progress-fill and --ease-progress-value.
Component
ease-progress-fill
Animated fill inside ease-progress. Set --ease-progress-value: 75%.
Utility
ease-text-gradient
Applies gradient text using primary color range.
Utility
ease-glass
Glassmorphism effect with backdrop-filter blur.
Utility
ease-truncate
Truncates overflowing text with ellipsis.
Utility
ease-sr-only
Visually hidden but accessible to screen readers.
