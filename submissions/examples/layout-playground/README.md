# Interactive Layout Playground

An interactive, live preview utility playground built to showcase Flexbox and Grid class helpers inside EaseMotion CSS.

## Features

- **Dynamic Modes**: Seamlessly switch between Flexbox and Grid layouts.
- **Flexbox Utility Controls**: Experiment with direction (`ease-flex-row`, `ease-flex-col`), alignment (`ease-justify-*`, `ease-items-*`), and wrap configurations (`ease-flex-wrap`).
- **Grid Utility Controls**: Configure columns (`ease-grid-cols-*`, `ease-grid-auto`) and alignments (`ease-place-*`).
- **Dynamic Content Scaling**: Add or remove layout children to see how column wrap and grid rows behave with varying content densities.
- **Dynamic Code Snippet Generation**: Watch class updates generate valid HTML structure in real time, and copy snippets with a single click.

## Utility Classes Demonstrated

### Flexbox
- `.ease-flex`
- `.ease-flex-row` / `.ease-flex-col`
- `.ease-justify-start` / `.ease-justify-center` / `.ease-justify-end` / `.ease-justify-between` / `.ease-justify-around` / `.ease-justify-evenly`
- `.ease-items-start` / `.ease-items-center` / `.ease-items-end` / `.ease-items-stretch`
- `.ease-flex-nowrap` / `.ease-flex-wrap` / `.ease-flex-wrap-reverse`

### Grid
- `.ease-grid`
- `.ease-grid-cols-1` / `.ease-grid-cols-2` / `.ease-grid-cols-3` / `.ease-grid-cols-4`
- `.ease-grid-auto`
- `.ease-place-start` / `.ease-place-center` / `.ease-place-end`

### Spacing (Gap)
- `.ease-gap-1` / `.ease-gap-2` / `.ease-gap-3` / `.ease-gap-4` / `.ease-gap-6` / `.ease-gap-8` / `.ease-gap-10`
