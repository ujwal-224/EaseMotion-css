# Sub-Orbital Space Tourism Booking Portal — Phase #856

A futuristic space tourism booking interface. Built entirely with **HTML5**, **CSS3**, and **vanilla JavaScript**. Part of the EaseMotion CSS design system showcase.

---

## Features

- Animated starfield with drifting layers and nebula
- Floating planet with parallax
- Live countdown timer to next launch
- Interactive destination cards with orbital animations
- Seat selection map with availability
- Dynamic booking form with summary
- Flight manifest with passenger status
- Weather display with launch readiness
- Warp speed effect on starfield
- Booking confirmation modal
- Responsive layout
- Reduced motion support
- Zero external dependencies
