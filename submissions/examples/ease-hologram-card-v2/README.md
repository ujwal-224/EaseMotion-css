# Ease Hologram Card v2

## Overview

Ease Hologram Card v2 is a futuristic card animation featuring animated scan lines, RGB border glow, holographic reflections, and smooth hover interactions using only HTML and CSS.

---

## Features

- 🌈 RGB animated border
- ✨ Holographic shine
- 📡 Moving scan line
- 🚀 Hover elevation
- 💎 Glassmorphism design
- 📱 Responsive layout
- ⚡ Pure HTML & CSS
