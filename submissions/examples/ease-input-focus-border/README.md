# Input Focus Border

An input with animated focus border — pure CSS.

## Features
- Smooth border color transition on focus
- Subtle box-shadow ring on focus
- Error state variant
- Clean dark theme styling

## Usage
```html
<input class="focus-input" placeholder="Text">
<input class="focus-input error" placeholder="Error">
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
