# Modal Scale Backdrop

A modal with scale-in animation and backdrop fade — pure CSS.

## Features
- Backdrop fade-in animation
- Modal scale-in with spring-like cubic-bezier
- Reusable modal component pattern

## Usage
```html
<div class="modal-backdrop">
  <div class="modal-box">
    <h3>Title</h3>
    <p>Content</p>
  </div>
</div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
