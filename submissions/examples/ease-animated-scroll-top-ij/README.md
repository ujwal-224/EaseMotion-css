# Ease Animated Scroll Top

A floating **Scroll to Top** button with a smooth bouncing animation. Clicking the button scrolls the page back to the top using smooth scrolling.

## Features

- Floating fixed-position button
- Smooth scroll-to-top behavior
- CSS bounce animation
- Responsive layout
- Pure HTML & CSS (minimal JavaScript for scrolling)

## Folder Structure

```
ease-animated-scroll-top-ij/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in your browser.
2. Scroll down the page.
3. Click the floating arrow button to smoothly return to the top.

## Technologies Used

- HTML5
- CSS3
- JavaScript
