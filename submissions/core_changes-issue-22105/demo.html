How It Works

Speed Multipliers

Each parallax element moves at a fraction of the scroll speed. A value of 0 means no movement, 1 means it scrolls at full speed (same as the page), and values above 1 move faster than scroll.

0 Static No parallax — scrolls with page
0.15 Slow ease-parallax (bg layer)
0.5 Medium Default parallax speed
0.8 Fast ease-parallax-fast
1.2 Reverse ease-parallax-reverse

What is Parallax?

Parallax scrolling creates an illusion of depth by moving background and foreground elements at different speeds as the user scrolls. This technique has been widely used in web design for storytelling, product showcases, and immersive landing pages.

Performance-First Approach

This implementation uses IntersectionObserver to only activate parallax on elements within the viewport — off-screen elements are ignored. All positioning is done via CSS transform: translateY() inside requestAnimationFrame for smooth 60fps updates without triggering layout recalculations.

Mobile Awareness

On devices with prefers-reduced-motion enabled, parallax automatically disables itself. Additionally, touch devices can optionally disable parallax to avoid scroll jank. Set data-ease-disable-mobile="true" on any element.

Integration with EaseMotion

Parallax pairs naturally with EaseMotion's existing reveal animations. Combine ease-reveal with ease-parallax for elements that both fade in and move at a different scroll speed — creating sophisticated scroll-driven experiences.

Implementation

Proposed JS Structure

// core/parallax.js — proposed structure
const parallaxElements = document.querySelectorAll('.ease-parallax, .ease-parallax-slow, .ease-parallax-fast, .ease-parallax-reverse');

const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.dataset.easeActive = 'true';
    } else {
      delete entry.target.dataset.easeActive;
    }
  });
});

function updateParallax() {
  parallaxElements.forEach(el => {
    if (el.dataset.easeActive !== 'true') return;
    const speed = parseFloat(el.dataset.easeSpeed) || 0.5;
    const rect = el.getBoundingClientRect();
    const offset = (rect.top - window.innerHeight) * speed;
    el.style.transform = `translateY(${offset}px)`;
  });
  requestAnimationFrame(updateParallax);
}

parallaxElements.forEach(el => observer.observe(el));
requestAnimationFrame(updateParallax);
