Theme Persistence Demo

Issue #12448 — The theme toggle button persists dark/light preference across page reloads using localStorage.

How It Works

  1. Inline script in <head> reads localStorage before rendering (prevents FOUC)
  2. If no saved preference, falls back to prefers-color-scheme
  3. Toggle button updates data-theme on <html> and saves to localStorage
  4. Toggle icon switches between sun (light) and moon (dark)

Test It

Click the toggle button (sun/moon) in the header. Reload the page — your preference is preserved!
