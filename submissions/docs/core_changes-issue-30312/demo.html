Cards (respond to theme)

Primary

var(--ease-text) / var(--ease-text-secondary)

Theme Classes

Use .ease-theme-dark or .ease-theme-light on any container to force a scheme.

.ease-theme-light

Always light

.ease-theme-dark

Always dark

Auto Mode

Add .ease-dark-mode-auto to <html> to follow prefers-color-scheme.

Add .ease-dark to force dark regardless of system preference.
